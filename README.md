# Discord-Music-Bot

Basic Discord bot for playing music in a Discord call.  The bot is made in Python and uses the discord.py library.

Features: 
- Queueing system
- Play
- Pause
- Resumem
- Stop

## Dependencies
`pip install discord.py`
`pip install python-dotenv`
`pip install yt-dlp`
`pip install PyNaCl`

You'll also need **FFmpeg** - Store the executable file (.exe) in this folder structure: *bin/ffmpeg/ffmpeg.exe*
FFmpeg can be downloaded here - https://www.ffmpeg.org/download.html

yt-dlp requires Node.js to be installed for the bot to work correctly. - https://nodejs.org/en/download
